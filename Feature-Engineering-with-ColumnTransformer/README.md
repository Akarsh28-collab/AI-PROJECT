# Feature Engineering using ColumnTransformer

This project demonstrates how to use `ColumnTransformer` in Scikit-Learn for feature engineering and preprocessing machine learning datasets.

---

## Project Overview

In this project, we perform:

- Numerical feature scaling using `StandardScaler`
- Categorical feature encoding using `OneHotEncoder`
- Data preprocessing using `ColumnTransformer`
- Conversion of transformed data into a clean DataFrame

This workflow is commonly used before training machine learning models.

---

## Technologies Used

- Python
- Pandas
- Scikit-Learn
- Jupyter Notebook
