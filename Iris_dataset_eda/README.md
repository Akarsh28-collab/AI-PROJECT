# 🌸 Iris Dataset EDA (Exploratory Data Analysis)

## 📌 Project Overview

This project performs **bivariate and multivariate exploratory data analysis (EDA)** on the Iris dataset using Pandas and Seaborn.

The goal is to understand relationships between features and identify patterns across different species.

---

## 📊 Dataset

* Source: Seaborn built-in dataset
* Total samples: 150
* Features:

  * Sepal Length
  * Sepal Width
  * Petal Length
  * Petal Width
* Target:

  * Species (Setosa, Versicolor, Virginica)

---

## 🔍 EDA Steps Performed

### 1. Basic Data Analysis

* Checked data types and structure
* Verified missing values
* Generated statistical summary

### 2. Bivariate Analysis

* Scatter plots to analyze relationships between features
* Box plots to compare feature distributions across species

### 3. Multivariate Analysis

* Pairplot to visualize relationships across all features
* Heatmap to identify feature correlations
* Distribution plots to understand feature spread

---

## 📈 Key Insights

* Petal length and petal width show strong correlation
* Setosa species is clearly separable from others
* Versicolor and Virginica have slight overlap
* Petal features are more important than sepal features

---

## 🛠️ Tools Used

* Pandas
* Seaborn

---



## 🚀 Conclusion

EDA helps in understanding data patterns and selecting important features before applying machine learning models.
