# 📜 Quotes Web Scraper

A simple Python web scraping project that extracts quotes, authors, and tags from [Quotes to Scrape](https://quotes.toscrape.com/) and stores the data in a CSV file.

---

## 🚀 Features

* Scrapes quotes from multiple pages
* Extracts:

  * 📝 Quote text
  * 👤 Author name
  * 🏷️ Tags
* Stores data in a structured CSV file
* Uses clean and beginner-friendly code

---

## 🛠️ Tech Stack

* Python 🐍
* requests
* BeautifulSoup (bs4)
* pandas

---
