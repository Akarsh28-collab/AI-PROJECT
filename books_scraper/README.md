# 📚 Books Scraper (Beginner Web Scraping Project)

This is a beginner-friendly web scraping project using Python.

## 🔧 Tech Stack
- Python
- Requests
- BeautifulSoup
- Pandas

## 📌 Features
- Scrapes book title, price, and rating
- Extracts data from multiple pages (1–16)
- Stores data in a Pandas DataFrame
- Exports data to CSV

## 🚀 Output
- books_data.csv

## 📚 Learning Outcome
- HTML parsing
- Web scraping basics
- Pagination handling
- Data cleaning (optional)

## 🌐 Source Website
http://books.toscrape.com/
